# Day 35 – Prompt Puzzle

## Project

🧩 Prompt Puzzle — Master AI Prompting Through Play

---

## Objective

Built an interactive Prompt Engineering game that teaches users how to design better AI prompts through gamified learning.

---

## Features

- Offline Single HTML App
- Drag and Drop Prompt Builder
- Prompt Cleaning Challenges
- Best Prompt Selection
- Live Scoring
- Timer
- Hints
- Accuracy Tracking
- Prompt Performance Report
- Replay with Random Scenarios
- Premium UI
- Responsive Design

## What I Learned

- Importance of clear instructions
- Role prompting improves AI quality
- Context dramatically affects responses
- Constraints reduce hallucinations
- Formatting makes outputs more consistent
- Better prompts produce better AI outputs
- Avoid over-engineering prompts
