# Day 33 – Media Integrity Analyzer

## Project

Built an interactive Media Integrity Analyzer using Claude AI.

## Features

- Theme Selection
- Headline Detective
- Emotion Detector
- Media Integrity Metrics
- Dashboard
- Replay

## Key Learnings

- Headlines often exaggerate information.
- Emotional language influences decisions.
- Source credibility matters.
- Verify information before sharing.
- Critical thinking helps detect manipulation.
