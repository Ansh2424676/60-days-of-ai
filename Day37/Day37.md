# Day 37 – Task Compass

## Project
Task Compass

## Objective

Learn how ownership, delegation, workflow, and collaboration function inside real organizations through an interactive management simulation.

---

## Features

- Drag and Drop
- Three Interactive Stages
- Organizational Thinking Dashboard
- Reflection Summary
- Offline HTML Application
- Dark Glassmorphism UI
- Responsive Design

## Key Learnings

- Ownership should remain clear to avoid confusion.
- Delegation improves workflow efficiency.
- Collaboration is essential for solving complex organizational problems.
- Most workplace challenges require coordinated efforts across multiple teams rather than a single individual.
