# Day 36 – Cognitive Pattern Explorer

## Objective

Built an offline psychology-inspired interactive web application using HTML, CSS and Vanilla JavaScript.

## Features

- Calm Mode
- Stress Mode
- Interactive Scenarios
- Drag and Drop Priorities
- Thinking Timeline
- Reflection Journal
- Progress Tracking
- Responsive Design
- Offline Support

## Key Learnings

- Different situations influence thinking patterns.
- Reflection is more useful than labeling.
- Interactive learning feels more engaging than traditional quizzes.
- Vanilla JavaScript can create rich offline applications.
- Reusable drag-and-drop components improve code organization.
