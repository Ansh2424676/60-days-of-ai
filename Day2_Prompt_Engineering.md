# Day 2 – Prompt Engineering

## What is Prompt Engineering?

Prompt Engineering is the process of giving AI clear context, goals, constraints, and expected output format so that it produces better and more reliable results.

A good prompt saves time, improves accuracy, and generates more useful responses.

## Weak Prompt

Help me learn Python.

### Problems:

* No goal
* No experience level
* No timeline
* No output format

## Engineered Prompt

Act as an experienced Python mentor.

I am a recent MCA graduate interested in becoming a Python Developer and Data Analyst.

Create a 30-day Python learning roadmap for me.

Requirements:

* Beginner to intermediate level
* Focus on real-world projects
* Include daily learning goals
* Recommend free learning resources
* Suggest practice exercises
* Include milestones for progress tracking

Output Format:

1. Week-wise roadmap
2. Skills to learn
3. Practice tasks
4. Mini projects
5. Final project recommendation

Goal:
Become job-ready for Python Developer and Data Analyst roles within 30 days.

## Key Learning

A normal prompt asks for information.

An engineered prompt provides:

* Context
* Role
* Goal
* Constraints
* Output format

This leads to significantly better AI responses.
