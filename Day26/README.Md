# Day 26 – Prior Authorization Workflow Simulator

## Overview

This project simulates the US Healthcare Prior Authorization (PA) workflow using an interactive drag-and-drop interface.

## Features

* Patient, Provider and Payer workflow lanes
* Multiple healthcare scenarios
* Medical necessity review
* Document collection process
* Prior authorization submission
* Approval, Pend and Denial outcomes
* Appeals and Peer-to-Peer review
* Progress tracker
* Days elapsed counter
* Efficiency score
* Approval celebration animation
* Responsive UI

## Technologies Used

* HTML
* CSS
* Vanilla JavaScript

## Learning Outcomes

* Understanding healthcare prior authorization workflows
* Workflow simulation design
* State management in JavaScript
* Drag-and-drop interactions
* Interactive educational UI development
