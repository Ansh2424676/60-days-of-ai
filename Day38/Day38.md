# Day 38 – Typing Speed Studio

## Project Overview

Typing Speed Studio is a premium browser-based typing practice platform developed using HTML, CSS, and JavaScript.

The application offers multiple typing modes, adaptive learning, real-time analytics, and session history without requiring any external libraries.

---

## Features

- Time Mode
- Word Count Mode
- Quote Mode
- Programming Mode
- Adaptive Mode
- Focus Mode
- Zen Mode
- Custom Text Mode
- Live WPM
- Accuracy
- CPM
- Mistake Counter
- Streak Tracking
- Analytics Dashboard
- Error Heatmap
- Performance Graph
- Session History
- Personal Best Tracking
- Achievement Badges
- Dark Theme
- Responsive Design

---

## Technologies Used

- HTML5
- CSS3
- JavaScript
